# kepler
the kepler keyboard. An ARM based, split, ergonomic keyboard

![current revision](./kepler-v0.3.jpg)

# features 

- OLED support (SSD1306 compatible)
- ARM based (STM32F303CCT6)
- QMK supported
- flippable PCB
- hotswappable switches (using kailh-mx sockets)
- 0402 components
